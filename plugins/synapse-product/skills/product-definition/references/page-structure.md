# 시냅스 기본 페이지 구조 상세

## 페이지 계층 구조

```
시냅스 플랫폼
├── 로그인 / 회원가입
│   └── 이메일 + 비밀번호 인증 (synapse.sh)
│
├── 워크스페이스 선택
│   ├── 소유/참여 워크스페이스 목록
│   └── 워크스페이스 생성
│
└── 워크스페이스 메인
    ├── 프로젝트 목록 (카드형)
    │   ├── 프로젝트 상세
    │   │   ├── 대시보드 (통계/진행 상황)
    │   │   ├── 워크샵 목록
    │   │   │   ├── 워크샵 상세
    │   │   │   │   ├── 태스크 목록
    │   │   │   │   ├── 작업 할당 설정
    │   │   │   │   └── 검수 설정
    │   │   │   └── 어노테이터 진입
    │   │   │       ├── 헤더
    │   │   │       ├── 툴바
    │   │   │       ├── 에디터 (메인 작업 영역)
    │   │   │       ├── 패널
    │   │   │       └── 설정/단축키
    │   │   ├── 데이터 컬렉션
    │   │   │   ├── 컬렉션 기본 설정
    │   │   │   ├── 데이터 구조 (파일 스펙)
    │   │   │   ├── 데이터 유닛 목록
    │   │   │   ├── 데이터 업로드
    │   │   │   ├── 데이터 익스포트
    │   │   │   └── 사용자 및 보안
    │   │   └── ML 실험
    │   │       ├── 실험 생성
    │   │       ├── 학습 실행/모니터링
    │   │       └── 모델 라이브러리
    │   └── 프로젝트 생성 마법사
    │
    ├── 카탈로그 (데이터셋 탐색)
    │
    ├── 스토리지 관리
    │   ├── 스토리지 연동 (S3, GCS, SFTP, Local)
    │   └── GPU 서버 연동
    │
    ├── 멤버 관리
    │   ├── 사용자 목록/초대
    │   ├── 역할 할당
    │   └── 권한 설정 (모듈별)
    │
    ├── 플러그인 관리
    │   ├── 플러그인 목록
    │   ├── 플러그인 설치/제거
    │   └── 플러그인 설정
    │
    └── 튜토리얼
        ├── 튜토리얼 목록
        ├── 튜토리얼 생성 (관리자)
        └── 튜토리얼 수행 (작업자)
```

## 카드형 UI 패턴

시냅스는 목록 페이지에서 **카드형 UI** 패턴을 일관되게 사용한다.

### 카드 구성 요소

- **썸네일**: 시각적 미리보기 (데이터 유형에 따라 다름)
- **제목**: 프로젝트/컬렉션/워크샵 이름
- **핵심 정보**: 상태, 진행률, 날짜 등
- **빠른 액션**: 카드에서 바로 수행 가능한 동작

### 대시보드 시각화

- 다양한 통계 데이터 카드
- 작업 상태별 진행 상황 차트
- 검수 상태 표시
- 데이터셋 분포/중복/유사도 차트

## 어노테이터 페이지 상세

### 이미지 어노테이터

| 영역 | 설명 |
|------|------|
| 헤더 | 과제 정보, 네비게이션, 저장/제출 버튼 |
| 툴바 | bbox, polygon, polyline, keypoint, segmentation 도구 |
| 에디터 | 이미지 표시 + 어노테이션 오버레이 |
| 패널 | 클래스 선택, 속성 편집, 어노테이션 목록 |

### 비디오 어노테이터

| 영역 | 설명 |
|------|------|
| 헤더 | 과제 정보, 네비게이션 |
| 에디터 | 비디오 플레이어 + 어노테이션 오버레이 |
| 타임라인 | 프레임 네비게이션, 트래킹 구간 표시 |
| 패널 | 객체 추적 설정, 클래스 선택 |

### 텍스트 어노테이터

| 영역 | 설명 |
|------|------|
| 에디터 | 텍스트/PDF/HTML 렌더링 + 하이라이트 |
| 패널 | NER 태그, 감정/의도 레이블, 관계 설정 |

### 3D 어노테이터

| 영역 | 설명 |
|------|------|
| 에디터 | 3D 포인트 클라우드 렌더링 (Multi-viewport) |
| 툴바 | Cuboid, 3D Segmentation 도구 |
| 패널 | 객체 속성, 클래스 선택 |

### 프롬프트 어노테이터

| 영역 | 설명 |
|------|------|
| 프롬프트 영역 | 사용자 정의 프롬프트 표시 |
| 응답 영역 | LLM 생성 응답 표시 |
| 평가 패널 | 응답 평가/수정/라벨링 |
